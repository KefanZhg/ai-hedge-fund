# Repo Agent Guide

This repository is configured for agent-assisted development with hardware safety constraints.

## Domain Coverage

- firmware C/C++
- Python/MATLAB/C++ tooling
- FPGA RTL and timing workflows
- PCB design and verification artifacts
- mechanical CAD interfaces and fabrication handoff docs
- lab automation code (read-only by default)

## Working Rules

- Keep production-impacting changes behind explicit approval.
- Use task worktrees for isolated changes.
- Record assumptions and validation evidence in task reports.
